Trabajo final de la asignatura Introduccion a las Ciencias de la Computacion y Progamacion en la universidad nacional de colombia de Colombia semestre 2021-2
Trabajo fue realizado por Augustro David Rico Dautt y Nicole Sofia Salazar Delgado
El trabajo consiste de un sudoku que se autogenera y que es valido ya que se confirma que tiene unica respuesta y utilizando la libreria pygames se genera un entorno grafico para rellenarlo
al momento de llenarlo correctamente se muestra un mensaje de felicitaciones y se puede guardar la puntuacion.
las mayores puntuaciones pueden ser consultadas en el menu de inicio.

Recomendaciones: el programa fue testeado en linux mint 20.3 utilizando python 3.9.5 por lo que se recomenda utilizar el mismo para su desarrollo o carga,
no obstante debe poder ejecutarse en windows de igual manera.

Requisitos: debe tenerse instalada la libreria 'Pygame' para poder ejecutar el programa, de esta libreria se utilizo la version 2.1.2 por lo que Se recomienda utilizar la misma.
para instalar la libreria 'Pygame' se recomienda utilizar el comando 'pip install -r requirements.txt' estando en la carpeta para instalar facilmente la version 2.1.2
